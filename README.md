# 🤖 Gemini Multimodal AI Workspace

A powerful, interactive multi-task AI dashboard built using **Streamlit** and the **Google Gemini API**. This workspace allows users to interact with advanced language and vision models through a clean, unified user interface.

## 🌟 Features
* **💬 Multi-Turn Chatbot:** A fully conversational assistant powered by Gemini Pro that maintains chat history seamlessly.
* **🖼️ Image Captioning Suite:** Upload any image (`PNG/JPEG`) and instantly generate descriptive context using vision capabilities.
* **⚡ Vector Text Embeddings:** Convert raw text strings into high-dimensional numeric vectors via `gemini-embedding-001`.
* **💡 Deep Q&A Module:** A comprehensive, knowledge-based utility to get rapid answers to complex prompts.

## 🛠️ Built With
* **Python 3.11+**
* **Streamlit** (UI Framework)
* **Google Generative AI SDK** (Gemini API)
* **Pillow (PIL)** (Image Processing)
